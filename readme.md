This is the backend of nlw-space applications

## Autor
- [@LeonardoGarciah](https://github.com/LeonardoGarciah)

## Command line

To create schemas run

```bash
  npx prisma migrate dev
```

To run project in dev mode

```bash
  npm run dev
```

To show prisma studio

```bash
  npx prisma studio
```
